# Baixa Música Flet

Aplicativo para download de músicas desenvolvido com Python e Flet.


## Download

1. Baixe a última versão;
2. Instale no diretório desejado;
3. Usufrua do aplicativo sem restrições e sem execuções extras;
4. Faça o download no link abaixo ⬇️

➡️ [Baixar última versão](https://github.com/Otavio-Barboza/Baixa_Musica_Flet/releases/latest)


## Tecnologias

- Python --- v(3.11.9)
- Flet --- v(0.28.3)
- yt-dlp --- v(2026.3.17)
- FFmpeg --- versão não específica


## Executando pelo código-fonte

Para executar o projeto a partir do código-fonte, é necessário instalar o FFmpeg e adicionar os executáveis na pasta:

```text
Assets/ffmpeg/
├── ffmpeg.exe
└── ffprobe.exe
```

- Link do FFmpeg para download na página do GitHub logo abaixo ⬇️

➡️ [Página do FFmpeg para download](https://github.com/FFmpeg/FFmpeg)
